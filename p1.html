Propiedades CSS

width :Establece la anchura de los elementos de bloque . Ejemplo : { width: 150px; }

max width : Limita la anchura máxima de los elementos de bloque . Ejemplo : (max-width: 400px)

height : Establece la altura de los elementos de bloque . Ejemplo : { height: 100px; }

max-height : Limita la altura máxima de los elementos de bloque . Ejemplo : { max-height: 300px; }

display : Establece el tipo de caja generada por un elemento . Ejemplo : [display: block]

overflow : Controla el comportamiento de los contenidos que no caben en su elemento contenedor . Ejemplo : [overflow: visible]

z-index : La propiedad CSS z-index indica el z-order de un elemento y sus descendientes. Ejemplo : z-index: auto | | inherit

line-height : Controlara con un valor el espacio entre líneas base de texto . Ejemplo : { line-height: 200% }

box-sizing : Se usa para modificar las propiedades por defecto del CSS box model . Ejemplo : box-sizing: content-box;
